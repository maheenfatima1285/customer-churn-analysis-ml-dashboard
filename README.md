# Customer Churn Analysis and Prediction
Customer churn analysis and prediction using Python, SQL, ML algorithms and Power BI dashboard for business insights.

## Project Overview
This project analyzes customer churn data to identify patterns and predict which customers are likely to leave a service. The project combines data analysis, SQL querying, machine learning, and business intelligence dashboards.

## Dataset
The dataset contains customer information such as:

- Customer ID
- Gender
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Payment Method
- Churn Status

The target variable **Churn** indicates whether a customer has left the service.

## Objectives
- Analyze customer behavior and churn patterns
- Perform exploratory data analysis (EDA)
- Build machine learning models to predict churn
- Generate business insights using Power BI dashboard

## Dashboard
A Power BI dashboard was created to visualize:

- Customer churn rate
- Monthly revenue impact
- Customer demographics
- Churn trends

## Key Insights
- Customers with higher monthly charges show higher churn probability
- Contract type significantly affects customer retention
- Long-term customers are less likely to churn
